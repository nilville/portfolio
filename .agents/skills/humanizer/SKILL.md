# Human Writing Transformer

## Description

Transforms AI-sounding text into writing that feels natural, individual, and genuinely authored by a person while preserving the original meaning, factual content, and intent.

## Activation

Use this skill whenever the user wants text rewritten, humanized, naturalized, personalized, conversationalized, or made less robotic.

## Core Objective

The goal is not merely to replace words.

The goal is to recreate how a thoughtful human would naturally express the same ideas.

Preserve:

* Meaning
* Facts
* Logic
* Tone requested by the user

Improve:

* Rhythm
* Flow
* Voice
* Readability
* Authenticity

## Multi-Pass Rewrite Process

### Pass 1: Meaning Extraction

Identify:

* Core message
* Essential facts
* Intended audience
* Desired tone

### Pass 2: Remove AI Patterns

Reduce:

* Formulaic transitions
* Predictable openings
* Generic conclusions
* Repetitive sentence structures

Avoid excessive use of:

* Furthermore
* Moreover
* Additionally
* In conclusion
* It is important to note
* Delve into
* Leverage
* Robust

### Pass 3: Rebuild Natural Flow

Use:

* Mixed sentence lengths
* Natural pacing
* Human rhythm
* Organic transitions

### Pass 4: Add Human Texture

When appropriate:

* Use contractions
* Prefer concrete language
* Vary paragraph lengths
* Use direct phrasing

### Pass 5: Voice Calibration

Match the user's intended style:

* Professional
* Academic
* Casual
* Technical
* Persuasive
* Personal

### Pass 6: Authenticity Review

Check for:

* Repeated wording
* Repeated sentence openings
* Uniform structure
* Overly polished phrasing

Revise if necessary.

## Output Rules

* Preserve meaning.
* Do not add facts.
* Do not explain the process.
* Output only the rewritten text unless analysis is requested.
